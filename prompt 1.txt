## 4. Development Roadmap (Modular Phases)

### Phase 1: Foundation & Refactoring (Target: Hari 1-2)

* [ ] Setup Git & GitHub Repo.
* [ ] Setup `.gitignore` agar data CSV tidak ter-upload.
* [ ] **Refactoring:** Memecah kode "Day 1" yang panjang menjadi 3 modul di folder `src/` (`data_loader.py`, `model_trainer.py`, `predictor.py`).
* [ ] Test run lokal.

### Phase 2: Model Improvement & Tracking (Target: Hari 3-4)

* [ ] Ganti Linear Regression dengan Random Forest / XGBoost.
* [ ] Integrasi **MLflow** untuk mencatat akurasi model setiap kali training.
* [ ] Simpan model (`.pkl`) dengan penamaan versi otomatis.

### Phase 3: Containerization (Target: Hari 5-6)

* [ ] Buat `Dockerfile`.
* [ ] Build Image Docker.
* [ ] Jalankan aplikasi dalam Container (Simulasi Server).

### Phase 4: CI/CD & Deployment (Target: Hari 7+)

* [ ] Setup GitHub Actions (Otomatis test kode saat push).
* [ ] Deploy ke Streamlit Cloud atau Cloud Server (AWS/GCP/Railway).
